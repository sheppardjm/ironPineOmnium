**Create src/components/Nav.astro:**

```astro
---
// Nav.astro — Sticky site navigation bar
// Rendered in BaseLayout.astro, outside .page-shell to avoid rise animation.
// Uses position: sticky (not fixed) to avoid layout flow problems.
// Logo uses <img> tag because logo.svg has hardcoded fills (cannot use currentColor).
---
<nav class="site-nav" aria-label="Site navigation">
  <div class="nav-inner">
    <a href="/" class="nav-logo" aria-label="Iron & Pine Omnium home">
      <img src="/logo.svg" alt="" width="52" height="39" />
    </a>
    <ul class="nav-links" role="list">

    </ul>

  </div>
</nav>
```

Note: The logo alt text is empty because the aria-label on the `<a>` tag provides the accessible name. Set both width="52" and height="39" (maintains ~4:3 ratio of the 1294x966 viewBox). The logo SVG has hardcoded grayscale fills — do NOT try to set fill via CSS.

```js
export default defineConfig({
  output: "static",
  vite: {
    plugins: [tailwindcss()],
  },
});
```

Note: Remove `fontProviders` from the import since it's no longer used.
