Note: The `<main>` tag gets `class="split-shell"` instead of `class="page-shell"`. The `.page-shell` wrapper is NOT needed here because the split layout is full-viewport. **Script changes:**

The `<script>` block logic stays exactly the same with ONE change to the error display:

Instead of:
```js
errorEl.hidden = false;
```
Use:
```js
errorEl.style.display = 'block';
```

And instead of:
```js
errorEl.hidden = true;
```
Use:
```js
errorEl.style.display = 'none';
```

And `errorEl.textContent` remains the same — it sets the banner text.

The error element is now `<div id="submit-error" class="submit-error-banner">` instead of `<p id="submit-error" class="submit-error" hidden>`. Update the TypeScript type cast accordingly (HTMLDivElement instead of HTMLParagraphElement).

**Remove the entire scoped `<style>` block.** All styles now come from global.css:
- `.split-shell`, `.split-brand`, `.split-form` — from Plan 01 global.css
- `.form-field input` — from Plan 01 global.css (borderless-bottom style)
- `.primary-button` — from Plan 01 global.css (dark button on light)
