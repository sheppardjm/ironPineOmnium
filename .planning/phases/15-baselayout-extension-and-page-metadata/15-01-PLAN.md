  <action>
Modify `src/layouts/BaseLayout.astro` to:

1. **Extend the Props interface** — add three optional props:
   ```typescript
   interface Props {
     title?: string;
     description?: string;
     ogImage?: string;   // path relative to site root, defaults to "/og-image.png"
     noindex?: boolean;   // when true, emit <meta name="robots" content="noindex">
     type?: string;       // og:type value, defaults to "website"
   }
   ```

2. **Destructure with defaults** in the frontmatter script:
   ```typescript
   const {
     title = "Iron & Pine Omnium",
     description = "A two-day gravel series in Michigan's Upper Peninsula combining day 1 moving time with day 2 sector speed and KOM points.",
     ogImage = "/og-image.png",
     noindex = false,
     type = "website",
   } = Astro.props;
   ```

3. **Construct absolute URLs** in the frontmatter script (after destructuring):
   ```typescript
   const canonicalURL = new URL(Astro.url.pathname, Astro.site);
   const ogImageURL = new URL(ogImage, Astro.site);
   ```

4. **Replace the existing `<link rel="icon" type="image/svg+xml" href="/logo.svg" />`** with the full favicon set:
   ```html
   <link rel="icon" type="image/x-icon" href="/favicon.ico" />
   <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
   <link rel="manifest" href="/site.webmanifest" />
   ```

5. **Add the following meta tag block** in `<head>`, AFTER the existing `<meta name="description">` and BEFORE `<link rel="preconnect">`:

   ```astro
   <!-- Canonical -->
   <link rel="canonical" href={canonicalURL} />

   <!-- Open Graph -->
   <meta property="og:type" content={type} />
   <meta property="og:url" content={canonicalURL} />
   <meta property="og:site_name" content="Iron & Pine Omnium" />
   <meta property="og:title" content={title} />
   <meta property="og:description" content={description} />
   <meta property="og:image" content={ogImageURL} />
   <meta property="og:image:width" content="1200" />
   <meta property="og:image:height" content="630" />
   <meta property="og:image:alt" content="Iron & Pine Omnium — two-day gravel weekend in Michigan's Upper Peninsula" />

   <!-- Twitter Card -->
   <meta name="twitter:card" content="summary_large_image" />
   <meta name="twitter:title" content={title} />
   <meta name="twitter:description" content={description} />
   <meta name="twitter:image" content={ogImageURL} />

   <!-- Conditional noindex -->
   {noindex && <meta name="robots" content="noindex" />}
   ```

6. **Keep the existing `<slot name="head" />`** at the end of `<head>` — it is still needed for Phase 16 JSON-LD injection.

Important notes:
- Do NOT add `twitter:site` — no event Twitter account exists.
- Do NOT use `set:html` for these tags — they are standard Astro expressions, not raw HTML strings.
- Ensure `canonicalURL` and `og:url` use the SAME variable so they can never mismatch.
- Do NOT duplicate the `<title>` or `<meta name="description">` tags — they already exist in the template.
