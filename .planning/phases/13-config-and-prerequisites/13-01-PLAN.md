<task type="auto">
  <name>Task 1: Install @astrojs/sitemap and configure astro.config.mjs</name>
  <files>astro.config.mjs, package.json, pnpm-lock.yaml</files>
  <action>
    1. Run `pnpm astro add sitemap` to install `@astrojs/sitemap` and auto-wire the integration into astro.config.mjs. This command adds the import and `integrations` array automatically.

    2. After the command completes, manually edit astro.config.mjs to add the `site` property and the sitemap `filter` option. The final file must look like this:

    ```javascript
    import { defineConfig } from "astro/config";
    import tailwindcss from "@tailwindcss/vite";
    import sitemap from "@astrojs/sitemap";

    export default defineConfig({
      site: "https://ironpineomnium.com",
      output: "static",
      integrations: [
        sitemap({
          filter: (page) =>
            page !== "https://ironpineomnium.com/submit-confirm/" &&
            page !== "https://ironpineomnium.com/error/",
        }),
      ],
      vite: {
        plugins: [tailwindcss()],
      },
    });
    ```

    Key details:
    - `site` must be exactly `"https://ironpineomnium.com"` (no trailing slash)
    - Filter URLs MUST include trailing slashes — the sitemap integration generates URLs with trailing slashes for static output
    - Exclude both `/submit-confirm/` and `/error/` — neither has SEO value
    - Do NOT change `output`, `vite`, or any other existing settings
    - If `pnpm astro add sitemap` produces a different structure for the integrations array, adjust to match the pattern above while preserving whatever the command set up
  </action>
  <verify>
    Run `pnpm build` and confirm:
    1. Build succeeds without errors
    2. `dist/sitemap-index.xml` exists
    3. `dist/sitemap-0.xml` exists and contains page URLs
    4. `dist/sitemap-0.xml` does NOT contain `submit-confirm` or `error`
    5. `dist/sitemap-0.xml` DOES contain `https://ironpineomnium.com/` (homepage)

    Commands:
    ```bash
    pnpm build
    cat dist/sitemap-index.xml
    cat dist/sitemap-0.xml
    grep -c "submit-confirm" dist/sitemap-0.xml  # should be 0
    grep -c "error" dist/sitemap-0.xml  # should be 0
    grep -c "ironpineomnium.com" dist/sitemap-0.xml  # should be >= 1
    ```
  </verify>
  <done>
    astro.config.mjs has `site: "https://ironpineomnium.com"` and `integrations: [sitemap({filter: ...})]`. Build produces sitemap-index.xml and sitemap-0.xml. submit-confirm and error pages are excluded from the sitemap. All other pages (index, submit, leaderboard, support) are included with absolute URLs.
