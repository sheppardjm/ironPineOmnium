   const eventSchema = {
     "@context": "https://schema.org",
     "@type": "Event",
     "name": "Iron & Pine Omnium",

2. **Add the JSON-LD script tag** as a direct child of `<BaseLayout>`, BEFORE the `<main>` element. This tag uses `slot="head"` to inject into BaseLayout's `<slot name="head" />` in `<head>`, and `set:html` to prevent HTML-escaping of JSON content:

   ```astro
   <script type="application/ld+json" set:html={JSON.stringify(eventSchema)} slot="head"></script>
   ```

   The resulting line in `index.astro` should look like:
   ```astro
   <BaseLayout
     title="Iron & Pine Omnium | Two Days of Gravel in the Hiawatha National Forest"
     description="A two-day gravel weekend in Michigan's Upper Peninsula — Hiawatha's Revenge fondo on Saturday, MK Ultra Gravel grinduro on Sunday. Submit your Strava activities for an overall result."
   >
     <script type="application/ld+json" set:html={JSON.stringify(eventSchema)} slot="head"></script>
     <main class="page-shell">
   ```

Important notes:
- Use `"@type": "Event"` (NOT `"SportsEvent"`) — research confirms Event is the only type documented as eligible for Google's event rich result cards. Add `"sport": "Cycling"` as a supplemental property for semantic accuracy.
- Use `set:html={JSON.stringify(eventSchema)}` — this is MANDATORY. Without `set:html`, Astro HTML-escapes the content producing `&quot;` entities that break JSON parsing. Google's crawler would silently ignore the block.
- Do NOT use curly-brace interpolation like `{JSON.stringify(eventSchema)}` as children — Astro renders this as a text node which also gets HTML-escaped.
- Do NOT fabricate a `streetAddress` — the event is on forest roads, not at a fixed venue. `addressLocality` + `addressRegion` + `addressCountry` is sufficient and honest.
- Use date-only format (`"2026-06-06"`) for startDate/endDate — no specific start time is known, and fabricating one violates Google's accuracy guidelines.
- Do NOT touch BaseLayout.astro — the `<slot name="head" />` already exists and requires no changes.
